The Chess Library is an open-source database of 700,000+ chess games, run by a single college student. It's currently in open beta testing right now so if you have any suggestions, bug findings, questions, or would like to help expand the database, please share using the donate link above. (Note: The back-end web scraping and data formatting scripts will be up soon.)
